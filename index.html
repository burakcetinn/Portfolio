Reminiscence

2d Platformer Game

In this platformer game, our main character is a witch who is trying to save her sister. She uses different element combinations to attack. Different elements have different impacts on the enemies. In this game, which consists of three stages, there are main enemies at the end of each stage. As you progress in the game, stages get more complicated and enemies are harder to defeat. There is also a story which you can follow with the memory fragments scattered around the platform. If you collect all the memories, you can achive the true ending.
